# Testing

## Included Tooling

- Vitest
- React Testing Library
- jsdom
- `@testing-library/jest-dom`

## What Is Covered

- reusable card rendering
- image fallback behavior
- URL construction in the data client
- route rendering smoke coverage

## Commands

```bash
npm run test
npm run test:watch
npm run typecheck
npm run lint
```
